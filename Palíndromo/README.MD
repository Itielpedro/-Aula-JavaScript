## PALÍNDROMO

Atividade proposta no curso de variáveis e tipos na linguagem JavaScript na plataforma Digital Innovation One.

## ATIVIDADE

Verifique, de duas maneiras diferentes entre si, se uma String é um palíndromo.
_Palíndromo: frase ou palavra que se pode ler, indiferentemente, da esquerda para a direita ou vice-versa (ex.: raiar, ama, ovo, radar)_
